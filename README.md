# Minesweeper

Game settings are input in the console and the game is played in a pop-up window. 
```
java Grid.java
```
<img width="1366" height="728" alt="image" src="https://github.com/user-attachments/assets/58645ac6-8e3f-4472-8ea7-48f343c95717" />

## Functionality

### Customization
<ul>
  <li>
    Option to choose grid layout and amount of bombs. </li>
    <li>Either default (10x10 w/ 25 bombs) or custom (with constraints).
  </li>
</ul>

### Game Over
<ul>
  <li>
    Losing with the option to play again (with same settings).
  </li>
  <li>
    Winning with the same aforementioned play again option.
  </li>
</ul>
